Αφού καταφέρατε να φθάσετε μέχρι εδώ σας αξίξουν πολλά συγχαρητήρια. Η επιμονή είναι ίσως το πιο βασικό χαρακτηριστικό της επίλυσης προβλημάτων και της εισαγωγής σας ως προγραμματιστής. Να θυμάστε [τα αποφθέγματα του Τόμας Έντισον](https://www.jcsl.com.cy/blog/business-quotes/thomas-edison), του εφευρέτη του ηλεκτρικού ρεύματος.

   _Το να είσαι ιδιοφυία είναι ένα τοις εκατό έμπνευση και ενενήντα εννέα τοις εκατό ιδρώτας. (Τόμας Έντισον)_
   
Κι από δω και πέρα τι; Είστε μόλις στην αρχή και θα πρέπει να αφιερώσετε πολύ περισσότερο χρόνο για να εντριφύσετε στη μαγεία του προγραμματισμού μαθαίνοντας καλά τουλάχιστο μια γλώσσα προγρ/σμού και λύνοντας ταυτόχρονα όσα περισσότερα προβλήματα μπορείτε.

Ο ιστοχώρος του Mathesis παρέχει αρκετά μαθήματα προγραμματισμού τα οποία μπορείτε να παρακολουθήσετε από δω και πέρα. Ας τα δούμε μαζί.

1. Αν έχετε κάποια επιχειρηματική ιδέα και θέλετε να δημιουργήσετε π.χ. μια ιστοσελίδα στο διαδίκτυο για να διαφημίσετε κάτι, τότε θα πρέπει οπωσδήποτε να γραφτείτε στο μάθημα [Εισαγωγή στην ανάπτυξη ιστοσελίδων με HTML5, CSS3, Javascript](https://mathesis.cup.gr/courses/course-v1:ComputerScience+CS3.1+20E/about).
1. Η Python είναι όπως είδαμε μια σχετικά εύκολη διερμηνευτική γλώσσα κατάλληλη τόσο για προγράμματα σεναρίων (scripts) όσο και για μεγάλες εφαρμογές. Με αυτήν μπορείτε να αυτοματοποιήσετε τη διαχείριση του συστήματός σας, να μάθετε να χακάρετε συστήματα ή να εισαχθήτε στον μαγικό κόσμο της Μάθησης της Μηχανής (Machine Learning) και πολλά άλλα. Αν ενδιαφέρεστε για την Python εγγραφείτε στα μαθήματα [Εισαγωγή στην Python](https://mathesis.cup.gr/courses/course-v1:ComputerScience+CS1.1+20C/about) και [Προχωρημένος προγραμματισμός με Python](https://mathesis.cup.gr/courses/course-v1:ComputerScience+CS1.2+20C/about).
1. Το Διαδίκτυο των Πραγμάτων, ή Internet of Things (IoT), μας επιτρέπει να προγραμματίζουμε έξυπνες συσκευές. Μελλοντικά θα μπορούμε να προγραμματίζουμε το ψυγείο μας ή την τηλεόρασή μας ώστε να κάνουν τη ζωή μας ευκολότερη. Εγγραφείτε στο μάθημα [Διαδίκτυο των Πραγμάτων (IoT) στην Πράξη: με Raspberry Pi και Python](https://mathesis.cup.gr/courses/course-v1:ComputerScience+CS4.1+20D/about) για να μάθετε περισσότερα.
1. Η Java είναι μια αντικειμενοστραφής μεταγλωττιζόμενη και πολύ διαδεδομένη γλώσσα προγραμματισμού κυρίως για εμπορικές εφαρμογές του διαδικτύου και όχι μόνο. Μπορείτε να μάθετε περισσότερα στο μάθημα [Εισαγωγή στην Java](https://mathesis.cup.gr/courses/course-v1:ComputerScience+CS2.1+19D/about).

Και θα ακολουθήσουν φυσικά και πολλά άλλα μαθήματα πληροφορικής από το Mathesis, οπότε φροντίστε να μένετε ενημερωμένοι. Καλή συνέχεια στο πολύ ενδιαφέρον αυτό ταξίδι που επιλέξατε.
